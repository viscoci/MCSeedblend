# SeedBlend

Server-side Minecraft mod: reseed an existing world for future terrain while preserving every already-generated chunk, with Minecraft's native terrain blending smoothing the boundary.

> **WORK IN PROGRESS** — see `OG_SPEC.md` for the full specification. Full README lands with the 0.1.0 release.

**⚠️ Always make a complete world backup before committing a reseed.**
